## 三、接口测试用例

### API-01：图片分析接口 - 正常请求

| 项目 | 内容 |
|------|------|
| 用例编号 | API-01 |
| 接口 | POST /api/detection/analyze |
| 请求体 | `{ "image": "data:image/jpeg;base64,/9j/4AAQ..." }` |
| 预期响应 | 200, `{ "success": true, "grade": 3, ... }` |
| 验证点 | success=true, grade 为 1-5 或 null, defects 对象完整, position 对象完整 |

### API-02：图片分析接口 - 无图片数据

| 项目 | 内容 |
|------|------|
| 用例编号 | API-02 |
| 接口 | POST /api/detection/analyze |
