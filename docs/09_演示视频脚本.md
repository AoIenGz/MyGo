## 四、录制注意事项

1. **提前准备**：确保后端服务和 Python 环境正常运行
2. **测试数据**：使用 13 张预设测试图片，确保每张都能正常检测
3. **模拟器设置**：DevEco Studio 模拟器分辨率设置为 1920×1080
4. **网络稳定**：确保前后端通信畅通，避免录制中断
5. **多次录制**：每个场景可单独录制，后期拼接
6. **旁白质量**：语速适中，咬字清晰，后期可添加字幕
7. **背景音乐**：选择轻柔的科技风格背景音乐，音量低于旁白
